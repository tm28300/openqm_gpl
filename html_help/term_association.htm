Association Term

 

Where the values stored in two or more multi-valued fields of the same table (file) are related such that corresponding values belong together, the fields are said to be associated.

 

An association is defined by giving it a name which appears in field 7 of the D or I-type dictionary entries of all fields in the association.  There should also be a phrase of the same name which lists the fields in the association in field 2.
