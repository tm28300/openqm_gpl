SET.DATE

    

 

The SET.DATE command adjusts the date used for all internal processing.

 

 

Format

 

SET.DATE date

 

where

 

dateis the new date to be used.

 

The SET.DATE command alters the internal date value affecting all application access to the date via the QMBasic DATE() function or the @DATE variable. It is useful when, for example, repeating month end processing after an error has been detected.
